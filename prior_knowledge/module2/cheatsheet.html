"Saturación" BJT vs MOSFET: BJT saturación = switch ON. MOSFET saturación = amplificador. MOSFET switch ON = región lineal/triodo. Nombres OPUESTOS.
P-channel high-side: GPIO LOW = ON (V_GS negativo → conduce). GPIO HIGH = OFF (V_GS = 0). Lógica invertida. En firmware: #define POWER_ON() GPIO_CLEAR(PIN)
¿Por qué carga entre VCC y Drain? Si va entre Source y GND, V_Source sube → V_GS se reduce → feedback negativo → MOSFET no satura. Carga arriba mantiene Source = GND → V_GS máximo.

2.2 Amplificadores Operacionales
Reglas de oro (con realimentación negativa) Entrevista
Regla 1: V+ = V− (diferencia = 0)
Regla 2: I+ = I− = 0 (impedancia entrada infinita)
Solo aplican con realimentación NEGATIVA (salida → V−). Sin realimentación → satura a ±V_supply (comparador).
Configuraciones Core

Inversor

V_out = −(R_f / R_in) × V_in
A_v = −R_f / R_in

V+ a GND. Invierte señal 180°. R_f = feedback, R_in = input.

No-inversor

V_out = (1 + R_f / R_in) × V_in
A_v = 1 + R_f / R_in

V_in entra por V+ (no-inversora). Ganancia ≥ 1.

Buffer (follower) Firmware

V_out = V_in (ganancia = 1)
R_f = 0 (cable), R_in = ∞ (abierto)

Desacopla impedancias. Sensor de alta Z → buffer → ADC. Uso diario en embedded.

Diferencial Aviónica

V_out = (R_f / R1) × (V2 − V1)

Alto CMRR — rechaza ruido común. Medir corriente con shunt en eVTOL.

Filtro activo pasa-bajos Aviónica
f_c = 1 / (2π × R_f × C)
Ej: R_f=10kΩ, C=1µF → f_c = 15.9 Hz
Filtro anti-aliasing antes del ADC. Obligatorio en DO-178C.
Método de análisis (4 pasos) Entrevista
  • 1. Verificar realimentación negativa (salida → V−)
  • 2. Aplicar V+ = V− (escribir voltaje conocido)
  • 3. Aplicar I = 0 → corriente por R_in = corriente por R_f
  • 4. Resolver para V_out
2.3 Conversión ADC/DAC ⭐
Fórmulas maestras Entrevista Core
LSB = V_ref / 2^N
V = código × LSB = código × (V_ref / 2^N)
código = V_in × 2^N / V_ref
f_s ≥ 2 × f_max (Nyquist-Shannon)
SNR_max = 6.02 × N + 1.76 dB
Resolución práctica (V_ref = 3.3V)
BitsNivelesLSBUso
825612.9 mVControl simple
101,0243.22 mVArduino
124,0960.806 mVCortex-M (estándar)

2416.7M0.197 µVAudio, termocuplas
Código máximo = 2^N − 1 → corresponde a V_ref − 1 LSB, NO a V_ref exacto.
"Resolución" ≠ "Precisión". ENOB (bits efectivos) < bits nominales por ruido.
Nyquist y Aliasing Entrevista
Nyquist: f_s ≥ 2 × f_max. En práctica: 5-10× f_max.
Aliasing: f_s < 2×f_max → frecuencia fantasma. f_alias = |f_s − f_señal|. IRREVERSIBLE después del ADC.
Filtro anti-aliasing: Pasa-bajos analógico ANTES del ADC. Elimina frecuencias > f_s/2.
El filtro pasa-bajos filtra FRECUENCIAS, no voltajes. Señal de 3V a baja freq → pasa. Señal de 0.1V a alta freq → bloqueada.
Cadena de adquisición Aviónica
Sensor → Buffer (op-amp) → Filtro anti-aliasing → ADC → Tu código C

SAR — búsqueda binaria Core
  • Compara V_in con V_ref/2 → MSB = 1 o 0
  • Ajusta rango a la mitad, compara de nuevo → siguiente bit
  • Repite N veces. Cada paso = 1 bit. 12 bits = 12 comparaciones.
  • DAC interno genera voltajes de prueba, comparador decide
  • El código final = suma de fracciones que dieron "SÍ": bit_k vale V_ref/2^k

2.4 Reguladores de Voltaje y Fuentes
LDO vs Switching Entrevista

LDO (Lineal)

P_calor = (V_in − V_out) × I_out
η = V_out / V_in
V_dropout típico: 100-300mV
  • ✓ Salida ultra-limpia (bajo ruido)
  • ✓ Simple, barato, sin EMI
  • ✗ Ineficiente si V_in >> V_out
  • ✗ Solo baja voltaje

Switching (Buck)

V_out = D × V_in (D = duty cycle)
η ≈ 85-95%
P_in = P_out / η
  • ✓ Eficiente (85-95%)
  • ✓ Puede subir Y bajar voltaje
  • ✗ Ruido (ripple, EMI)
  • ✗ Layout PCB crítico
"Switching siempre es mejor" → FALSO. Para ADC de precisión, el ruido del switching corrompe mediciones. Usar LDO después del switching para limpiar.

2.5 Ruido e Integridad de Señal — EMI/EMC
Definiciones clave Core
SNR = 20 × log₁₀(V_signal / V_noise) [dB]
ADC N bits: SNR_max = 6.02N + 1.76 dB
12 bits → ~74 dB máx. Si circuito tiene 40dB → últimos ~6 bits son ruido
  • EMI: Ruido que tu circuito genera
  • EMC: Funcionar bien en presencia de EMI y no generar EMI excesivo
  • DO-160G: Estándar de tests EMC para aviación

Técnicas de mitigación Entrevista Core

Capacitor de desacoplo Entrevista

  • 100nF cerámico en cada pin VCC de cada IC. Sin excepciones.
  • 10µF por rail de alimentación
  • Lo más cerca posible del pin (milímetros)
  • Reserva local de carga para transitorios

Ground plane

  • Capa entera de cobre = GND
  • Baja Z de retorno, blindaje entre capas
  • NO cortar el ground plane
  • PCB 4 capas: Señales/GND/Power/Señales

Separación analógico-digital

  • Zonas físicamente separadas en PCB
  • AGND + DGND → un solo punto de unión
  • Star ground bajo el ADC

Oversampling (firmware) Firmware

ruido_reducido = ruido / √N
16 muestras → ruido/4 → +2 bits

NO reemplaza el filtro anti-aliasing analógico.

Cadena completa de protección
Blindaje → Filtro anti-alias → Desacoplo 100nF → GND plane → ADC → FW oversample
DO-160G define tests de EMI/EMC obligatorios. Tu equipo debe sobrevivir EMI externo Y no generar EMI excesivo.
